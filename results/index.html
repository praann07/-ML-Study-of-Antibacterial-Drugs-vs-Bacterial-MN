IFPTML Analysis Results

Machine Learning Study of Antibacterial Activity Against E. coli DNA Gyrase

Study Overview

This analysis applies the IFPTML methodology (Information Fusion based on Perturbation Theory and Machine Learning) to predict antibacterial activity of compounds targeting Escherichia coli DNA gyrase enzyme. The study combines molecular descriptors, perturbation analysis, and metabolic network topology to build robust predictive models.

Methodology Highlights

  • Dataset: 1,033 compounds from ChEMBL database with DNA gyrase inhibition data
  • Activity Threshold: IC50 ≤ 1000 nM classified as "Active"
  • Features: 41 IFPTML descriptors (11 base + 11 perturbation + 11 normalized + 8 network)
  • Models: K-Nearest Neighbors (KNN) and Logistic Regression
  • Evaluation: 75/25 train-test split with comprehensive metrics

1. Activity Distribution Analysis

What This Shows

The left panel displays the distribution of compounds classified as active vs inactive based on the 1000 nM IC50 threshold. The right panel shows the distribution of IC50 values on a logarithmic scale, with the red dashed line indicating the activity threshold.

Key Observations

  • The dataset contains approximately 700 active compounds and 350 inactive compounds
  • Class imbalance ratio of roughly 2:1 favoring active compounds
  • IC50 values span several orders of magnitude, demonstrating compound diversity
  • Peak concentration of compounds near the threshold indicates critical potency range

2. Molecular Descriptor Distributions

What This Shows

Six key molecular descriptors are compared between active (green) and inactive (red) compounds. These physicochemical properties help distinguish effective DNA gyrase inhibitors from less potent compounds.

Key Observations

  • Molecular Weight (MolWt): Active compounds tend toward moderate molecular weights (400-600 Da)
  • LogP: Active compounds show favorable lipophilicity distribution for cell membrane penetration
  • TPSA: Active compounds have lower topological polar surface area, aiding cellular uptake
  • H-bond Donors/Acceptors: Active compounds generally have fewer H-bond donors (2-3), optimal for bioavailability
  • Ring Count: Active compounds contain 3-4 rings, typical of heterocyclic antibiotics

3. Descriptor Correlation Matrix

What This Shows

This correlation heatmap reveals relationships between molecular descriptors. Strong correlations (red) indicate features that provide similar information, while weak or negative correlations (blue) suggest complementary features.

Key Observations

  • High correlation (0.76-0.84): MolWt, TPSA, NumHAcceptors, and RingCount are strongly correlated, reflecting molecular size
  • Negative correlation (-0.37): LogP inversely correlates with TPSA and NumHAcceptors (hydrophobic vs hydrophilic balance)
  • Independent features: NumHDonors shows weak correlation with most features, providing unique information
  • Feature redundancy suggests opportunities for dimensionality reduction

4. E. coli Metabolic Network Topology

What This Shows

This visualization represents the Escherichia coli metabolic network structure, where nodes represent metabolites and edges represent enzymatic reactions. Network topology features (degree distribution, density, centrality) are integrated into the IFPTML feature vector.

Key Observations

  • The network exhibits scale-free topology with hub metabolites (highly connected nodes)
  • DNA gyrase operates within a complex metabolic context, affecting multiple pathways
  • Network features capture systems-level effects of enzyme inhibition
  • Integration of network topology improves prediction by considering metabolic consequences

5. ROC Curves - Model Performance

What This Shows

Receiver Operating Characteristic (ROC) curves plot the true positive rate against false positive rate at various classification thresholds. The Area Under Curve (AUC) quantifies overall model discrimination ability.

Key Observations

  • KNN Performance: AUC = 0.841 - Excellent discrimination capability
  • Logistic Regression: AUC = 0.667 - Good but lower than KNN
  • Both models significantly outperform random guessing (dashed diagonal, AUC = 0.5)
  • KNN captures non-linear relationships in IFPTML feature space more effectively
  • High AUC values validate the IFPTML methodology for antibacterial prediction

6. Confusion Matrices

What This Shows

Confusion matrices display the breakdown of predictions: true positives (active correctly identified), true negatives (inactive correctly identified), false positives (inactive misclassified as active), and false negatives (active misclassified as inactive).

Key Observations

  • KNN: 151 true positives, 51 true negatives, 20 false negatives, 37 false positives
  • Logistic Regression: 159 true positives, 16 true negatives, 12 false negatives, 72 false positives
  • KNN achieves better balance between sensitivity and specificity
  • Logistic Regression tends to over-predict active compounds (higher false positive rate)
  • Low false negative rates in both models minimize missing potentially active drugs

7. Comprehensive Model Comparison

What This Shows

This bar chart compares five key performance metrics across both models: Accuracy, Precision, Recall, F1-Score, and ROC-AUC. Higher values indicate better performance.

Key Observations

  • KNN outperforms Logistic Regression across all metrics
  • Accuracy: KNN (0.779) vs LR (0.676) - KNN correctly classifies 78% of compounds
  • Precision: KNN (0.803) vs LR (0.688) - KNN has higher confidence in positive predictions
  • Recall: KNN (0.883) vs LR (0.930) - Both models effectively identify active compounds
  • F1-Score: KNN (0.841) vs LR (0.791) - KNN achieves better precision-recall balance
  • ROC-AUC: KNN (0.841) vs LR (0.667) - KNN superior at ranking compounds by activity probability

8. Feature Importance Analysis

What This Shows

This horizontal bar chart displays the top 20 most influential features from the Logistic Regression model, ranked by coefficient magnitude. Larger bars indicate stronger influence on antibacterial activity prediction.

Key Observations

  • Top Predictors: HeavyAtomCount and its perturbation/normalized variants dominate, indicating molecular size is critical
  • Molecular Weight: MolWt and related features (Delta_MolWt, ZScore_MolWt) appear prominently
  • Lipophilicity: LogP perturbation features suggest deviation from average lipophilicity impacts activity
  • Structural Features: Ring count, CSP3 fraction, and aromatic rings contribute to discrimination
  • IFPTML Value: Perturbation (Delta_) and normalization (ZScore_) features provide additional predictive power beyond base descriptors
  • The diversity of important features validates the multi-faceted IFPTML approach

Study Conclusions

The IFPTML methodology successfully integrates molecular descriptors, perturbation theory, and metabolic network topology to predict antibacterial activity against E. coli DNA gyrase with high accuracy.
